Apache ZooKeeper on OS X in Five Minutes or Less

« Previous
/
Next »
By ypai / February 23, 2013 / Apache ZooKeeper, Distributed Systems / 3 comments

This is a quick guide to running Apache ZooKeeper on OS X. Basic knowledge of getting around in the terminal is assumed. Hope it’s helpful!

Download Apache ZooKeeper

Release bundles are available at the ZooKeeper downloads page.

Unpack the Bundle

The following will install ZooKeeper at ~/opt/zookeeper but you can install it anywhere you want.

Create directory if it doesn’t exist:

$ mkdir ~/opt

Extract the archive to target directory:

$ tar xf zookeeper-3.4.5.tar.gz -C ~/opt

Create a soft-link for convenience:

$ ln -s ~/opt/zookeeper-3.4.5 ~/opt/zookeeper

Copy Sample Configuration File

This is the two minute drill, but find more details in the ZooKeeper documentation under configuration options.

$ cd ~/opt/zookeeper
$ cp conf/zoo_sample.cfg conf/zoo.cfg

Start ZooKeeper

Go to where you installed ZooKeeper:

$ cd ~/opt/zookeeper

Run without server administration script arguments to see available options:

$ bin/zkServer.sh
JMX enabled by default
Using config: /opt/zookeeper/bin/../conf/zoo.cfg
Usage: bin/zkServer.sh {start|start-foreground|stop|restart|status|upgrade|print-cmd}

Start ZooKeeper:

$ bin/zkServer.sh start

Check ZooKeeper is Running Correctly

Telnet to the ZooKeeper port and issue one of ZooKeeper’s four letter words, you should see similar output:

$ telnet localhost 2181

Explore ZooKeeper using the Command Line Client

Run the command line client to explore the nodes and data in ZooKeeper:

$ cd ~/opt/zookeeper
$ bin/zkCli.sh -server localhost:2181
